# Image Downloader

Простое расширение для Google Chrome, которое позволяет массово скачивать изображения по списку прямых ссылок. 

## Установка

Расширение не опубликовано в магазине расширений Chrome, поэтому вы можете установить его вручную в режиме разработчика:

1. Скачайте папку `_extension` в виде ZIP-архива и распакуйте её.
2. Откройте браузер Chrome и перейдите на страницу расширений: `chrome://extensions/`.
3. Включите **Режим разработчика** в правом верхнем углу.
4. Нажмите кнопку **Загрузить распакованное расширение** слева вверху.
5. Выберите папку, в которую вы распаковали файлы репозитория.

## Использование

1. Нажмите на иконку расширения на панели инструментов Chrome.
2. Вставьте список прямых ссылок на изображения (каждая ссылка с новой строки).
3. Укажите расширение скачиваемых изображений. 
4. Нажмите кнопку **Скачать**.

Файлы будут загружены в папку "Downloads" с именами формата `image_001.jpg`, `image_002.jpg` и т. д. 

Если для доступа к изображению по ссылке необходима авторизация, то авторизуйтесь перед использованием расширения.
